### Hey, I'm Justin 👋

Senior software architect with 20+ years building software across .NET, SQL, TypeScript, and the messy middle where legacy systems meet modern services.

I'm currently at [Veson Nautical](https://veson.com), working across platform architecture, analytics, services, and long-lived business systems. Before that, I spent years at [Volition](https://en.wikipedia.org/wiki/Volition_(company)) — the studio behind Saints Row and Red Faction — where I developed a habit that never really left: find the broken manual process, understand it deeply, then automate it out of existence.

#### What I'm building now

Lately, most of my energy is going into **agentic workflows**: systems where AI agents can plan, execute, validate, and carry context forward instead of acting like disposable chat windows.

My main personal project is **Observatory** — a cognitive infrastructure layer for managing context, memory, briefings, session continuity, and multi-agent orchestration across interfaces. The goal is simple: make the system around the work smart enough that momentum doesn't keep getting lost.

Some current threads:

- 🔄 **Autonomous improvement loops** — agents that evaluate, diagnose, and refine their own skills and tool definitions
- 🧠 **Personal knowledge systems** — semantic search, context ledgers, memory, and continuity across tools
- 🏗️ **Specification-driven development** — structured workshops that turn intent into testable specs before implementation
- 🛰️ **Agent observability** — making agent behavior inspectable, debuggable, and improvable instead of magical
